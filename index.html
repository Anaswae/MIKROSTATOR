MIKROSTATOR

Aplikasi yang digunakan untuk manajemen hotspot pada Router Mikrotik, alternatif menarik bagi Router yang tidak support atau tidak bisa dipasang User Manager.

Anda bisa memantau banyak router sekaligus dalam satu halaman, generate voucher, menambahkan batas pemakaian (limit-uptime) menggunakan nominal uang, transfer menit pemakaian (limit-uptime) antar user dan lihat hasil penjualan pada laporan penjualan.

Download V1 View on Github

Monitor More Router

Pantau semua Router Mikrotik anda dalam satu halaman tanpa harus bolak balik berpindah halaman. Manfaatkan fitur 'Quick Connect' jika anda tidak ingin informasi login Router Mikrotik anda atau manfaatkan PIN Security jika anda ingin menyimpannya ke DB MIKROSTATOR.

Generate Voucher

Buat voucher sebanyak-banyaknya, desain sendiri voucher anda dengan template editor dan nikmati hasil penjualannya setiap pelanggan anda memakai voucher tersebut.

Simple Billing

Tambahkan batas waktu pemakaian hotspot pelanggan anda dengan fitur billing pada MIKROSTATOR yang memungkinkan anda menambahkannya sesuai harga perjam layanan hotspot anda.

Selling Report

Pantau hasil penjualan voucher dan billing anda setiap waktu. Laporan bisa anda Download dan rekap sebagai dokumen pribadi anda.

Realtime Monitor

Manfaatkan fitur 'Auto Refresh' untuk memonitor router anda dengan waktu yang anda bisa setel sesuai keinginan anda.

Pantau Dimana Saja

MIKROSTATOR bisa dipasang di layanan hosting dan sudah teruji berjalan normal dan lancar. Bahkan anda bisa memasangnya di Android menggunakan aplikasi web server PHP (teruji menggunakan aplikasi Penguin sebagai web servernya).
